# Drachendekoration-Home-Assistant
Bei der Suche nach einer neuen Zimmerdekoration bin ich auf das Modell von @thatcaddad (https://makerworld.com/de/models/876292-mounted-dragon-head#profileId-829309) gestoßen. Mit einigen Anpassungen sowie einer Basis habe ich das ganze für mein Regal als Dekoration gedruckt.

Mit einem ESP32 habe ich das ganze in Home assistant eingebunden.

Komponenten:
